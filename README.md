## Work Out Timer App
This app contains  two json work outs and  each file  has list of exercises.

## FEATURE OF THE APP
Edit exercise  name

Edit exercises start time and end time

Save exercises changes  to local storage

Timer to keep track of the exercise


## STATE MANAGEMENT

## LOCAL STORAGE
